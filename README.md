Calculadora de Média Aritmética
Este projeto é uma calculadora simples de média aritmética, onde o usuário pode inserir quatro números e calcular a média desses números. O projeto é desenvolvido usando tecnologias Front End, incluindo HTML, CSS e JavaScript.

Descrição
A aplicação consiste em uma página web interativa que permite ao usuário inserir quatro valores numéricos e calcular a média aritmética desses valores. A interface é simples e intuitiva, e o cálculo é realizado diretamente no navegador usando JavaScript.

Tecnologias Utilizadas
HTML: Estrutura básica da página e dos elementos de entrada.
CSS: Estiliza a página para uma melhor aparência e usabilidade.
JavaScript: Implementa a lógica de cálculo da média e manipula a interface do usuário.
Estrutura do Projeto
O projeto é composto por três arquivos principais:

index.html

Define a estrutura da página e os elementos do formulário de entrada.
Inclui referências ao CSS para estilos e ao JavaScript para funcionalidades.
styles.css

Define o estilo da página, incluindo layout, cores, fontes e botões.
Torna a interface mais agradável e responsiva.
script.js

Contém a lógica para calcular a média aritmética dos quatro números inseridos.
Manipula a DOM para exibir o resultado na página.
Como Funciona
HTML:

O arquivo index.html define a estrutura básica da página, incluindo um formulário com quatro campos de entrada para números e um botão para realizar o cálculo.
O resultado da média é exibido em um elemento div logo abaixo do formulário.
CSS:

O arquivo styles.css estiliza a página para melhorar a apresentação.
O layout é centralizado e o formulário é estilizado para ser mais atraente e fácil de usar.
JavaScript:

O arquivo script.js contém a função calcularMedia(), que é acionada quando o usuário clica no botão "Calcular Média".
A função obtém os valores dos campos de entrada, calcula a média e exibe o resultado no elemento div com o id resultado.
Uso
Abra o arquivo index.html em um navegador web.
Insira quatro números nos campos fornecidos.
Clique no botão "Calcular Média".
A média aritmética dos quatro números será exibida abaixo do formulário.
